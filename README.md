# Sales Dashboard

A modern, scalable dashboard built with Next.js 15, TypeScript, and Tailwind CSS.

## Atomic Design Implementation
This project follows Atomic Design principles:
- **Atoms**: Basic UI elements like `Button.tsx` and `Input.tsx`.
- **Organisms**: Complex components like `SalesChart.tsx` using Recharts.
- **Pages**: Layouts like `dashboard/page.tsx` that orchestrate data and components.

## Setup Instructions
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the development server.
4. Open [http://localhost:3000/dashboard](http://localhost:3000/dashboard).
